# 🌿 Flora Marchetti – Website de Paisagismo

Website institucional desenvolvido para a **Flora Marchetti**, empresa especializada em **paisagismo e jardinagem em Curitiba e Região Metropolitana**.

O objetivo do projeto é apresentar os serviços da empresa de forma moderna, responsiva e otimizada para **SEO**, permitindo que clientes encontrem facilmente informações e entrem em contato.

---

# 📷 Preview

O site possui as seguintes seções principais:

* Hero (apresentação da empresa)
* Quem Somos
* Serviços de Paisagismo
* Contato
* Redes Sociais

Interface limpa, com foco em **experiência do usuário, performance e otimização para buscadores**.

# 🔎 SEO (Otimização para Buscadores)

O projeto foi desenvolvido com foco em **SEO para negócios locais**, incluindo:

### Meta Tags

* Title otimizado
* Meta description
* Keywords
* Canonical URL

### Open Graph

Permite melhor visualização ao compartilhar o site em redes sociais como:

* Facebook
* WhatsApp
* Instagram

### Schema.org

Implementação de **dados estruturados** utilizando JSON-LD para ajudar mecanismos de busca a entender melhor o site.

Tipo utilizado:

```
HomeAndConstructionBusiness
```

Informações estruturadas incluem:

* Nome da empresa
* Área de atuação
* Localização
* Telefone
* Website
